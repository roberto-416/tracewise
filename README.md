# Tracewise

Static transit reference site. Pilot city: Osaka.

Makes complex urban transit systems legible from the outside — stopping patterns, frequencies, through-running, and interchange structure.

**Live site:** https://roberto-416.github.io/tracewise/

## Structure

```
assets/     CSS, JS data loader, map config
data/       JSON data files (cities + per-city subdirs)
tools/      Python build scripts (not deployed)
```
